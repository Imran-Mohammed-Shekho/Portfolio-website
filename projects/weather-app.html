Weather App

A real-time weather app built with Flutter for city search and current-location lookup. The project emphasizes API-driven data display, clear status presentation, and fast user feedback.

Overview

This project provides current weather information through a simple mobile interface. Users can search by city or rely on device location to get weather details quickly.

It is a solid example of combining a Flutter front end with external API data while keeping the UI approachable and fast to understand.

Project details

Key Features

Weather information delivered with minimal friction.

City search

Users can look up weather by city without needing to navigate through complex screens.

Current location

The app can fetch weather based on the user’s location for a faster default experience.

Live weather data

Temperature, humidity, wind, and condition values are shown using OpenWeather data.

Refresh flow

Users can update the view and retrieve fresh data without losing the clean layout.

Responsive design

The design adapts well across device sizes while keeping the forecast information readable.

Dark mode styling

The UI remains comfortable to use in darker environments and gives the app a more finished feel.
