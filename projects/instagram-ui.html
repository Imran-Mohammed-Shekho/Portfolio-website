Overview

This project focuses on interface accuracy more than backend behavior. The goal was to recreate the visual structure of an Instagram profile screen using Flutter widgets and strong spacing control.

It demonstrates comfort with stacked layouts, reusable components, and interface composition that feels native on mobile devices.

Key Features

A profile screen recreated with reusable Flutter widgets.

Profile header

User image, name, bio, and profile metadata are laid out to closely match the original pattern.

User stats

Post, followers, and following counts are styled with strong hierarchy and balance.

Edit profile action

The action row is structured to feel like a complete profile screen instead of a static mockup.

Story highlights

Highlight bubbles demonstrate horizontal content and small reusable display components.

Post grid

A media grid completes the profile composition and reinforces consistent spacing decisions.

Responsive layout

The screen adapts to modern phone sizes, including the iPhone-style mockups in the gallery.
