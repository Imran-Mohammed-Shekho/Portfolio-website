Overview

The BMI Calculator project was built around a simple but useful health interaction: entering body information, calculating BMI, and returning understandable category feedback.

The value of the project is in its clear user flow, polished styling, and the way it turns a straightforward calculation into a refined mobile experience.

Key Features

A simple health tool with clean feedback loops.

BMI calculation

The app calculates body mass index from user inputs quickly and clearly.

Clear input handling

Height and weight are organized in a way that keeps the calculation flow easy to follow.

Category feedback

Results map back to understandable health categories instead of leaving users with only a number.

Smooth UI details

The app uses motion and spacing to make a utility flow feel more polished.

Responsive screens

The layout adapts to different mobile sizes while keeping the calculation surface stable.

Dark mode

Dark mode helps the utility feel more complete and comfortable during regular use.
