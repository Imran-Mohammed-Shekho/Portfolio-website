Overview

Todoey was built as a small productivity app and UI exercise. The priority was to make adding, viewing, and removing notes feel immediate without overloading the screen with extra actions.

It is one of the clearest examples in the portfolio of using a simple product idea to practice responsive layout, state updates, and polished interaction flow.

Key Features

A minimal app built for speed and clarity.

Instant note creation

Users can add notes quickly without navigating through extra screens or forms.

Easy deletion

Removing items stays simple so the interface always feels lightweight and useful.

Real-time UI refresh

Changes are reflected immediately, reinforcing a fast and responsive app feel.

Responsive layout

The layout remains readable across phone and larger device sizes.

Dark mode

A dark theme helps the product feel polished even as a small utility application.

Lightweight experience

The interface stays fast because the product scope is intentionally focused and simple.
