Alef Learning App

Overview

Alef is a polished educational UI prototype focused on teaching the Kurdish alphabet through visual repetition, guided lesson order, and child-friendly presentation.

The app starts with an animated splash and welcome flow, then moves into a bottom-navigation shell with a home screen and profile screen. Learners open alphabet lessons one by one, see example words and images, and unlock the next letter as they progress.

It currently uses local content only. There is no backend, database, or API yet, and progress is stored in memory, which means lesson completion resets when the app restarts.

Key Features

Designed to make early Kurdish alphabet learning visual and engaging.

Sequential alphabet lessons

Letters unlock one by one, giving learners a simple guided path instead of a confusing open list.

Word and image pairing

Each letter is supported with a Kurdish example word and matching illustration for visual repetition.

Animated learning flow

Splash screen, welcome flow, transitions, and decorated cards keep the experience lively.

Colorful themed UI

The app uses gradients, custom theming, local illustrations, and decorated lesson cards.

Provider-based state

Theme switching and lesson progression are managed through dedicated providers in Flutter.

Dark mode support

The visual system includes dark mode and custom Kurdish-script font support for local content.

App flow

Animated splash screen, welcome page, bottom navigation shell, then lesson browsing from home.

Data model

All letters, colors, categories, words, and images are stored locally inside the app package.

Next improvements

Persist learner progress, store theme settings, and replace static profile activity with real data.
